1.Реализовать веб-приложение на PHP, которое позволит пользователям:
- Зарегистрироваться.
- Войти в систему с использованием логина и пароля.
- Выйти из системы.
- После входа видеть приветственное сообщение с именем пользователя.

Технологии - PHP, MySQL, HTML/CSS, можно использовать фреймворки/паттерны.

Результат:
- Архив с исходным кодом или GitHub.
- Краткая инструкция по запуску (структура базы, подключение).

2. На основе задания 1 реализовать REST API для управления пользователями.
Задача:
На основе базы данных и логики из первого задания реализовать REST API, которое позволяет:
- Добавлять нового пользователя.
- Получать список всех пользователей.
- Получать данные конкретного пользователя по его id.
- Редактировать данные пользователя (имя,email).
- Удалять пользователя.

Авторизация для API через API KEY.
Запросы:
GET /users — получить список пользователей.
GET /users/{id} — получить данные пользователя по id.
POST /users — создать нового пользователя.
PUT /users/{id} — обновить все данные пользователя.
PATCH /users/{id} — обновить часть данных пользователя (например, только email).
DELETE /users/{id} — удалить пользователя.

В запросе и ответе данные передаются в формате json.

Технологии - PHP, MySQL, можно использовать фреймворки/паттерны.

Результат:
- Архив с исходным кодом или GitHub.
- Краткая инструкция по запуску (эндпоинты, пример запроса curl).
